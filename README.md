# 🚀 AeroGIS Pro v4  
### Aerial Survey Intelligence Platform

AeroGIS Pro is a **zero-dependency, single-file geospatial intelligence system** designed for:

- ✈️ Drone & aerial survey planning  
- 🌍 GIS analysis and mapping  
- 🌦 Weather-integrated flight optimization  
- 🛰 Environmental and land-use modelling  

Built for engineers, surveyors, and planners across **Kenya and East Africa**.

---

# 🌟 Key Highlights

- 📦 **Single-file application (index.html)** — no installation required  
- ⚡ **Zero dependencies** — no Node.js, no backend, no build tools  
- 🌐 Runs directly in any modern browser  
- 🛰 Integrated satellite imagery and terrain models  
- 📊 Full **Environmental Planning Lab (7-step workflow)**  
- 📡 Live weather integration (Open-Meteo API)  
- 📁 Multiple export formats (GeoJSON, KML, Mission formats)

# 🧪 Quick Start (First Mission)

### Step 1 — Select Survey Area
- Use predefined shapes or draw custom polygon  

### Step 2 — Choose Platform & Sensor
- Select drone and camera  
- System auto-configures parameters  

### Step 3 — Set Flight Parameters
- Altitude  
- Overlap  
- Field of View  
- Wind direction  

### Step 4 — Generate Flight Plan
- Automatic flight lines appear  
- Waypoints generated  

### Step 5 — Export & Analyze
- Export data  
- Review environmental outputs
